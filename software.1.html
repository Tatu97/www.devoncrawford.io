MY GOAL

I have been developing software since 2015 with a deep understanding of core computer science principles ranging from low-level algorithms and data structures in C, to object-oriented design in Java, Arduino and embedded software, assembly code, and obviously web design. I have a keen eye for automating processes especially in the multimedia field as I am both a software developer and a filmmaker/photographer. Combining this knowledge I have developed multimedia tools such as a Timelapse plugin for Adobe Photoshop, batch processing of YouTube video descriptions, and embedded software to control the circuitry of physical technology. All the work on this website are personal projects I developed in my spare time, which you may use for free and view all the source code on github! I have a very entrepreneurial spirit and enjoy solving meaningful problems in a simplistic and lightweight approach.

TECHNICAL SKILLS

JAVA

My first programming language.

Java is my go to for object oriented design with 3 years of experience in my personal projects. Most of my multimedia tools are written in Java, as well as graphics interface applications. Recently I have been extending the YouTube API to implement missing features and autonomous processes, as well as Photoshop Plugins and an A* pathfinding visualization. I will continue using Java for agile development at a high level. Learn more about my Java project development here.

C

Low-level design excites me because you become very close to the hardware and can exploit significant performance improvements. I am very interested in constructing libraries and data structures at a low-level, as well as transforming low level constructs from assembly to C and implementing high level constructs from C to Java and so on. I enjoy the concept of manual memory management and use it to my advantage, bringing significant performance improvements in the design of algorithms.
